## Layout [#layout]

Import the parts, and compose them together.

```tsx title="segmented-input-usage.tsx"
import {
  SegmentedInput,
  SegmentedInputItem,
} from "@/components/ui/segmented-input";

return (
  <SegmentedInput>
    <SegmentedInputItem placeholder="First" />
    <SegmentedInputItem placeholder="Second" />
    <SegmentedInputItem placeholder="Third" />
  </SegmentedInput>
);
```
