## Usage

---

```tsx title="swap-imports.tsx"
import { Swap, SwapOff, SwapOn } from "@/components/ui/swap";
```

```tsx title="swap-snippet.tsx"
<Swap>
  <SwapOn>
    <Sun className="h-8 w-8" />
  </SwapOn>
  <SwapOff>
    <Moon className="h-8 w-8" />
  </SwapOff>
</Swap>
```

### Accessibility

- When `activationMode` is `"click"`, the root element has `role="button"` and `aria-pressed` attributes
- Supports keyboard navigation with Enter and Space keys (click mode)
- Respects `prefers-reduced-motion` by disabling animations
- Includes `aria-disabled` when disabled
