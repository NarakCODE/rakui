### Styling

The Status component uses Tailwind CSS with CVA (class-variance-authority) for variant management. Each variant includes:

- **Border color** - Subtle border with opacity
- **Background color** - Light tinted background
- **Text color** - Matching variant color
- **Indicator color** - The animated dot inherits the variant color

The `StatusIndicator` includes:
- `before:` pseudo-element with `animate-ping` for the ripple effect
- `after:` pseudo-element for the solid dot center
