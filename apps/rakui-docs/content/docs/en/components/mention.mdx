## Layout [#layout]

Import the parts, and compose them together.

```tsx title="mention-usage.tsx"
import {
  MentionRoot,
  MentionLabel,
  MentionInput,
  MentionPortal,
  MentionContent,
  MentionItem,
} from "@/components/ui/mention";

return (
  <MentionRoot>
    <MentionLabel />
    <MentionInput />
    <MentionPortal>
      <MentionContent>
        <MentionItem />
      </MentionContent>
    </MentionPortal>
  </MentionRoot>
)
```

## API Reference [#api-reference]

The container for all mention parts. Mention tags can be styled using the `data-tag` attribute within the root.

An accessible label that describes the mention input. Associates with the input element for screen readers.
