## Layout [#layout]

Import the parts, and compose them together.

```tsx title="marquee-usage.tsx"
import {
  Marquee,
  MarqueeContent,
  MarqueeEdge,
  MarqueeItem,
} from "@/components/ui/marquee";

return (
  <Marquee>
    <MarqueeContent>
      <MarqueeItem>Item 1</MarqueeItem>
      <MarqueeItem>Item 2</MarqueeItem>
    </MarqueeContent>
    <MarqueeEdge side="left" />
    <MarqueeEdge side="right" />
  </Marquee>
);
```

## Usage [#usage]

### Basic Marquee

A simple horizontal marquee that scrolls items from right to left.

<ComponentTabs name="marquee-demo" />

### Pause on Hover

Add the `pauseOnHover` prop to pause the animation when the user hovers over the marquee.

```tsx title="marquee-snippet.tsx"
<Marquee pauseOnHover>
  <MarqueeContent>
    {items.map((item) => (
      <MarqueeItem key={item.id}>{item.name}</MarqueeItem>
    ))}
  </MarqueeContent>
</Marquee>
```

### Pause on Keyboard

Add the `pauseOnKeyboard` prop to allow users to pause/resume the animation by pressing the spacebar.

```tsx title="marquee-snippet.tsx"
<Marquee pauseOnKeyboard>
  <MarqueeContent>
    {items.map((item) => (
      <MarqueeItem key={item.id}>{item.name}</MarqueeItem>
    ))}
  </MarqueeContent>
</Marquee>
```

### Vertical Orientation

Use `side="top"` or `side="bottom"` for vertical marquees.

```tsx title="marquee-snippet.tsx"
<Marquee side="top" className="h-[400px]">
  <MarqueeContent>
    {items.map((item) => (
      <MarqueeItem key={item.id}>{item.name}</MarqueeItem>
    ))}
  </MarqueeContent>
  <MarqueeEdge side="top" />
  <MarqueeEdge side="bottom" />
</Marquee>
```

### Auto Fill

Use the `autoFill` prop to automatically duplicate content to fill the container.

```tsx title="marquee-snippet.tsx"
<Marquee autoFill>
  <MarqueeContent>
    {items.map((item) => (
      <MarqueeItem key={item.id}>{item.name}</MarqueeItem>
    ))}
  </MarqueeContent>
</Marquee>
```

### Custom Speed

Adjust the animation speed with the `speed` prop (pixels per second).

```tsx title="marquee-snippet.tsx"
<Marquee speed={100}>
  <MarqueeContent>
    {items.map((item) => (
      <MarqueeItem key={item.id}>{item.name}</MarqueeItem>
    ))}
  </MarqueeContent>
</Marquee>
```

### ARIA Attributes

- The marquee has `role="marquee"` and `aria-live="off"` to indicate it's a decorative scrolling element
- When `pauseOnKeyboard` is enabled, the marquee is focusable and has visible focus styles
- Items within the marquee can contain their own interactive elements
