## Usage

```tsx title="stat-demo.tsx"
import { ArrowUpRight, DollarSign } from "lucide-react";
import {
  Stat,
  StatDescription,
  StatIndicator,
  StatLabel,
  StatTrend,
  StatValue,
} from "@/components/ui/stat";

export function RevenueStat() {
  return (
    <Stat>
      <StatLabel>Monthly revenue</StatLabel>
      <StatValue>$48,240</StatValue>
      <StatIndicator color="success" variant="icon">
        <DollarSign />
      </StatIndicator>
      <StatTrend trend="up">
        <ArrowUpRight />
        12.4% from last month
      </StatTrend>
      <StatDescription>
        Compared with the previous billing cycle.
      </StatDescription>
    </Stat>
  );
}
```

## Examples

### Indicator Variants

`StatIndicator` supports `default`, `icon`, `badge`, and `action` variants so the same layout can cover dashboards, scorecards, and navigational summaries.

<ComponentTabs name="stat-indicator-demo" />

### With Supporting Details

Use `StatSeparator` and `StatDescription` when the card needs extra context without changing the overall grid structure.

```tsx title="stat-snippet.tsx"
import {
  Stat,
  StatDescription,
  StatIndicator,
  StatLabel,
  StatSeparator,
  StatTrend,
  StatValue,
} from "@/components/ui/stat";

<Stat>
  <StatLabel>Active customers</StatLabel>
  <StatValue>1,284</StatValue>
  <StatIndicator color="info" variant="badge">
    +86
  </StatIndicator>
  <StatTrend trend="up">7.1% week over week</StatTrend>
  <StatSeparator />
  <StatDescription>
    New workspace invites outpaced churn by 4.3x this week.
  </StatDescription>
</Stat>;
```

### Layout Notes

The root uses a two-column grid. Labels and values stay in the first column, indicators render in the second column, and the trend, separator, and description rows span the full card width.
