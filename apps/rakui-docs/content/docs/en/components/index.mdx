## Overview

All components share the following characteristics:

- **Accessible** — Built with ARIA standards and keyboard navigation support
- **Customizable** — Fully themeable with CSS variables and class overrides
- **Type-safe** — Written in TypeScript with full type definitions
- **Performant** — Optimized for minimal bundle size and fast rendering
