## Layout

Import the parts, and compose them together.

```tsx title="masonry-usage.tsx"
import {
  Masonry,
  MasonryItem,
} from "@/components/ui/masonry";

return (
  <Masonry>
    <MasonryItem />
  </Masonry>
)
```
