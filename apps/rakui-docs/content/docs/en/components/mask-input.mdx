## Usage [#usage]

### Basic Masks

The Mask Input component comes with several built-in masks for common use cases.

<ComponentTabs name="mask-input-demo" />

### Built-in Masks

| Mask | Pattern | Description |
|------|---------|-------------|
| `phone` | `(###) ###-####` | Phone number with area code |
| `ssn` | `###-##-####` | Social Security Number |
| `date` | `##/##/####` | Date in MM/DD/YYYY format |
| `time` | `##:##` | Time in HH:MM format |
| `creditCard` | `#### #### #### ####` | Credit card number |
| `creditCardExpiry` | `##/##` | Credit card expiry MM/YY |
| `zipCode` | `#####` | 5-digit ZIP code |
| `zipCodeExtended` | `#####-####` | 9-digit ZIP+4 |
| `currency` | `$###,###.##` | Currency with formatting |
| `percentage` | `##.##%` | Percentage value |
| `licensePlate` | `###-###` | License plate format |
| `ipv4` | `###.###.###.###` | IPv4 address |
| `macAddress` | `##:##:##:##:##:##` | MAC address |
| `isbn` | `###-#-###-#####-#` | ISBN-13 format |
| `ein` | `##-#######` | Employer Identification Number |

### Currency Masks

Support for international currency formatting with locale-aware symbols and separators.

```tsx title="mask-input-snippet.tsx"
// US Dollar (default)
<MaskInput mask="currency" placeholder="$0.00" />

// Euro with German locale
<MaskInput
  mask="currency"
  currency="EUR"
  locale="de-DE"
  placeholder="0,00 €"
/>

// Japanese Yen
<MaskInput
  mask="currency"
  currency="JPY"
  locale="ja-JP"
  placeholder="¥0"
/>
```

### With Form Integration

Integration with react-hook-form and zod validation.

```tsx title="mask-input-snippet.tsx"
import { zodResolver } from "@hookform/resolvers/zod";
import { useForm } from "react-hook-form";
import * as z from "zod";
import { MaskInput } from "@/components/ui/mask-input";

const formSchema = z.object({
  phone: z.string().min(10, "Phone number must be at least 10 digits"),
});

function MyForm() {
  const form = useForm({
    resolver: zodResolver(formSchema),
  });

  return (
    <FormField
      control={form.control}
      name="phone"
      render={({ field }) => (
        <FormItem>
          <FormLabel>Phone Number</FormLabel>
          <FormControl>
            <MaskInput
              mask="phone"
              value={field.value}
              onValueChange={(_masked, unmasked) => field.onChange(unmasked)}
            />
          </FormControl>
          <FormMessage />
        </FormItem>
      )}
    />
  );
}
```
