## API Reference

### AvatarGroup

The main avatar group container that handles layout and masking of child elements.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `max` | `number` | - | Maximum number of avatars to show before displaying overflow count |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout orientation |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of all child avatars |
| `renderOverflow` | `(count: number) => React.ReactNode` | - | Custom renderer for the overflow indicator |
| `className` | `string` | - | Additional CSS classes to apply |

### Avatar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of the avatar |
| `className` | `string` | - | Additional CSS classes to apply |

### AvatarImage

Extends all props from `@radix-ui/react-avatar` Image component.

### AvatarFallback

Extends all props from `@radix-ui/react-avatar` Fallback component.

## Styling

### Custom Colors

You can customize the fallback background color using Tailwind CSS classes.

```tsx title="avatar-group-snippet.tsx"
import { Avatar, AvatarFallback } from "@/components/ui/avatar";
import { AvatarGroup } from "@/components/ui/avatar-group";

export function CustomColors() {
  return (
    <AvatarGroup max={3}>
      <Avatar>
        <AvatarFallback className="bg-blue-100 text-blue-700">A</AvatarFallback>
      </Avatar>
      <Avatar>
        <AvatarFallback className="bg-green-100 text-green-700">B</AvatarFallback>
      </Avatar>
      <Avatar>
        <AvatarFallback className="bg-purple-100 text-purple-700">C</AvatarFallback>
      </Avatar>
    </AvatarGroup>
  );
}
```

### Adding Borders

Add borders to avatars for a different visual style.

```tsx title="avatar-group-snippet.tsx"
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar";
import { AvatarGroup } from "@/components/ui/avatar-group";

export function WithBorders() {
  return (
    <AvatarGroup max={3}>
      <Avatar className="ring-4 ring-background">
        <AvatarImage src="https://github.com/shadcn.png" />
        <AvatarFallback>CN</AvatarFallback>
      </Avatar>
      <Avatar className="ring-4 ring-background">
        <AvatarImage src="https://github.com/ethanniser.png" />
        <AvatarFallback>EN</AvatarFallback>
      </Avatar>
      <Avatar className="ring-4 ring-background">
        <AvatarFallback>+1</AvatarFallback>
      </Avatar>
    </AvatarGroup>
  );
}
```

### Status Indicators

Add status indicators to show online/offline status.

```tsx title="avatar-group-snippet.tsx"
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar";
import { AvatarGroup } from "@/components/ui/avatar-group";

export function WithStatus() {
  return (
    <AvatarGroup max={3}>
      <div className="relative">
        <Avatar>
          <AvatarImage src="https://github.com/shadcn.png" />
          <AvatarFallback>CN</AvatarFallback>
        </Avatar>
        <span className="absolute bottom-0 right-0 h-2.5 w-2.5 rounded-full border-2 border-background bg-green-500" />
      </div>
      <div className="relative">
        <Avatar>
          <AvatarImage src="https://github.com/ethanniser.png" />
          <AvatarFallback>EN</AvatarFallback>
        </Avatar>
        <span className="absolute bottom-0 right-0 h-2.5 w-2.5 rounded-full border-2 border-background bg-yellow-500" />
      </div>
      <Avatar>
        <AvatarFallback>+1</AvatarFallback>
      </Avatar>
    </AvatarGroup>
  );
}
```
