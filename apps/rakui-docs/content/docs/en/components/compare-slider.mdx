## Layout

---

Import the parts and compose them together.

```tsx title="compare-slider-snippet.tsx"
import {
  CompareSlider,
  CompareSliderBefore,
  CompareSliderAfter,
  CompareSliderHandle,
} from "@/components/ui/compare-slider";

export default function Example() {
  return (
    <CompareSlider className="h-64 w-full">
      <CompareSliderBefore label="Before">
        <img src="/image-before.jpg" alt="Before" />
      </CompareSliderBefore>
      <CompareSliderAfter label="After">
        <img src="/image-after.jpg" alt="After" />
      </CompareSliderAfter>
      <CompareSliderHandle />
    </CompareSlider>
  );
}
```

## Theming

### Container Theming

Use `data-slot="compare-slider"` to style the root container:

```tsx title="compare-slider-snippet.tsx"
<CompareSlider className="rounded-lg border-2 border-border shadow-lg" />
```

### Before/After Theming

Style the before and after containers:

```tsx title="compare-slider-snippet.tsx"
<CompareSliderBefore className="bg-linear-to-r from-blue-500 to-purple-500" />
<CompareSliderAfter className="bg-linear-to-r from-green-500 to-teal-500" />
```

### Handle Theming

Customize the handle appearance:

```tsx title="compare-slider-snippet.tsx"
<CompareSliderHandle className="w-12 bg-primary/80 rounded-full" />
```

### Label Theming

Style the labels positioned on each side:

```tsx title="compare-slider-snippet.tsx"
<CompareSliderLabel className="bg-black/70 text-white font-bold" />
```

### ARIA Attributes

- Root element has `role="slider"` with `aria-valuemin`, `aria-valuemax`, and `aria-valuenow`
- `aria-orientation` indicates horizontal or vertical orientation
- Before/After elements have `role="img"` with `aria-labelledby` when labels are present
- Handle has `role="presentation"` and `aria-hidden="true"`

## Features

- Drag and hover interaction modes
- Horizontal and vertical orientations
- Smooth pointer-based dragging with proper event handling
- Full keyboard navigation support
- Customizable handle with default directional icons
- Optional labels with automatic positioning
- Touch-optimized for mobile devices
- Works with images, videos, or any content
- Controlled and uncontrolled state management
