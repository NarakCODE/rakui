## Layout [#layout]

Import the parts, and compose them together.

```tsx title="listbox-usage.tsx"
import {
  Listbox,
  ListboxGroup,
  ListboxGroupLabel,
  ListboxItem,
  ListboxItemIndicator,
} from "@/components/ui/listbox";

return (
  <Listbox>
    <ListboxGroup>
      <ListboxGroupLabel>Group Label</ListboxGroupLabel>
      <ListboxItem>
        Item Label
        <ListboxItemIndicator />
      </ListboxItem>
    </ListboxGroup>
  </Listbox>
);
```
