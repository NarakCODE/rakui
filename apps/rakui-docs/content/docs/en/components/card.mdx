## API Reference

### Card

The main container component for the card.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "transparent" \| "secondary" \| "tertiary"` | `"default"` | Visual style variant of the card |
| `size` | `"default" \| "sm"` | `"default"` | Size of the card padding |
| `className` | `string` | - | Additional CSS classes to apply |

### CardHeader

The header section of the card, typically containing title and description.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes to apply |

### CardTitle

The title element within the card header.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes to apply |

### CardDescription

The description element within the card header.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes to apply |

### CardContent

The main content area of the card.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes to apply |

### CardFooter

The footer section of the card, typically containing actions.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes to apply |

## Styling

### Variant Styles

The Card component uses CVA (class-variance-authority) for variant management:

- **Default**: `bg-card text-card-foreground ring-foreground/10`
- **Transparent**: `bg-transparent text-foreground ring-transparent`
- **Secondary**: `bg-secondary text-secondary-foreground ring-foreground/10`
- **Tertiary**: `bg-muted/40 text-foreground/90 ring-foreground/10`

The CardFooter adapts its background color based on the card variant using `group-data-[variant=variant-name]` selectors.

### Custom Colors

```tsx title="card-snippet.tsx"
import { Card, CardHeader, CardTitle, CardContent } from "@/components/ui/card";

export function CustomCard() {
  return (
    <Card className="bg-gradient-to-br from-blue-500 to-purple-600 text-white border-0">
      <CardHeader>
        <CardTitle>Custom Styled Card</CardTitle>
      </CardHeader>
      <CardContent>
        <p>This card uses custom gradient background.</p>
      </CardContent>
    </Card>
  );
}
```
