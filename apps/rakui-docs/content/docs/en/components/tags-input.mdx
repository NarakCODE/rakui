## Layout [#layout]

Import the parts, and compose them together.

```tsx title="tags-input-usage.tsx"
import {
  TagsInput,
  TagsInputClear,
  TagsInputInput,
  TagsInputItem,
  TagsInputLabel,
  TagsInputList,
} from "@/components/ui/tags-input";

return (
  <TagsInput>
    <TagsInputLabel />
    <TagsInputList>
      {tags.map((tag) => (
        <TagsInputItem key={tag} value={tag}>
          {tag}
        </TagsInputItem>
      ))}
      <TagsInputInput placeholder="Add tag..." />
    </TagsInputList>
    <TagsInputClear>Clear</TagsInputClear>
  </TagsInput>
);
```

## Usage [#usage]

### Basic

A simple tags input with editable tags, a label, and a clear button.

<ComponentTabs name="tags-input-demo" />

### Editable

Enable the `editable` prop to allow inline editing of existing tags. Use `addOnPaste` to create multiple tags from pasted text.

<ComponentTabs name="tags-input-editable-demo" />

### With Validation

Use `onValidate` to reject invalid tags and `onInvalid` to show feedback. Combine with `max` to limit the number of tags.

<ComponentTabs name="tags-input-validation-demo" />

### With Sortable

Combine with the `Sortable` component to let users reorder tags via drag and drop.

<ComponentTabs name="tags-input-sortable-demo" />
