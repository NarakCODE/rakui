## Layout

Import the parts and compose them together.

```tsx title="components/ui/action-bar.tsx"
import {
  ActionBar,
  ActionBarSelection,
  ActionBarSeparator,
  ActionBarGroup,
  ActionBarItem,
  ActionBarClose,
} from "@/components/ui/action-bar";

<ActionBar open={open} onOpenChange={setOpen}>
  <ActionBarSelection>{selectedCount} selected</ActionBarSelection>
  <ActionBarSeparator />
  <ActionBarClose>
    <X className="h-4 w-4" />
  </ActionBarClose>
  <ActionBarSeparator />
  <ActionBarGroup>
    <ActionBarItem onSelect={handleCopy}>
      <Copy className="h-4 w-4" />
      Copy
    </ActionBarItem>
    <ActionBarItem variant="destructive" onSelect={handleDelete}>
      <Trash2 className="h-4 w-4" />
      Delete
    </ActionBarItem>
  </ActionBarGroup>
</ActionBar>;
```

## Usage

### Basic Usage

The Action Bar appears at the bottom of the viewport when items are selected.

```tsx title="action-bar-basic.tsx"
<ActionBar open={selectedIds.length > 0} onOpenChange={setOpen}>
  <ActionBarSelection>{selectedIds.length} selected</ActionBarSelection>
  <ActionBarSeparator />
  <ActionBarGroup>
    <ActionBarItem onSelect={handleDelete}>Delete</ActionBarItem>
  </ActionBarGroup>
</ActionBar>
```

### With Close Button

Include a close button to allow users to dismiss the action bar.

```tsx title="action-bar-with-close.tsx"
<ActionBar open={open} onOpenChange={setOpen}>
  <ActionBarSelection>3 selected</ActionBarSelection>
  <ActionBarSeparator />
  <ActionBarClose>
    <X className="h-4 w-4" />
  </ActionBarClose>
  <ActionBarSeparator />
  <ActionBarGroup>
    <ActionBarItem onSelect={handleAction}>Action</ActionBarItem>
  </ActionBarGroup>
</ActionBar>
```

### Custom Positioning

Use the `className` prop to customize the position of the action bar.

```tsx title="action-bar-positioning.tsx"
{
  /* Bottom Left */
}
<ActionBar open={open} onOpenChange={setOpen} className="left-6 translate-x-0">
  {/* ... */}
</ActionBar>;

{
  /* Bottom Right */
}
<ActionBar
  open={open}
  onOpenChange={setOpen}
  className="right-6 left-auto translate-x-0"
>
  {/* ... */}
</ActionBar>;

{
  /* Top Center */
}
<ActionBar open={open} onOpenChange={setOpen} className="top-6 bottom-auto">
  {/* ... */}
</ActionBar>;
```

## Accessibility

The Action Bar follows the [WAI-ARIA Toolbar pattern](https://www.w3.org/WAI/ARIA/apg/patterns/toolbar/).

### ARIA Attributes

- The Action Bar has `role="toolbar"` and `aria-orientation="horizontal"`
- Action groups have `role="group"` for semantic grouping
- The close button has `aria-label="Close"` for screen readers
- Separators have `aria-hidden="true"` to exclude from accessibility tree
