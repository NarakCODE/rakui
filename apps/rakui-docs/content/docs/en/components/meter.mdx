## Usage

---

```tsx title="meter-basic.tsx"
import { Meter } from "@/components/ui/meter";

export function StorageMeter() {
  return <Meter label="Storage" value={65} />;
}
```

## Examples

### Multiple Meters

Display multiple meters together to show resource usage at a glance.

<ComponentTabs name="meter-demo" />

### Value Boundaries

Values outside the 0-100 range are automatically clamped to the valid range.

```tsx title="meter-boundaries.tsx"
import { Meter } from "@/components/ui/meter";

export function MeterBoundaries() {
  return (
    <div className="flex flex-col gap-4">
      <Meter label="Underflow" value={-10} />
      <Meter label="Overflow" value={150} />
      <Meter label="Exact Zero" value={0} />
      <Meter label="Exact Full" value={100} />
    </div>
  );
}
```

### Custom Labels

Use any string as the label to describe the measured resource.

```tsx title="meter-labels.tsx"
import { Meter } from "@/components/ui/meter";

export function MeterLabels() {
  return (
    <div className="flex flex-col gap-4">
      <Meter label="Disk Usage" value={73} />
      <Meter label="Network Bandwidth" value={45} />
      <Meter label="GPU Memory" value={92} />
    </div>
  );
}
```

## Accessibility

- The underlying `Progress` component uses the `progressbar` role for screen reader support.
- The label and percentage text are rendered as visible text, providing context without relying solely on ARIA attributes.
