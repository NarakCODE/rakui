## Layout [#layout]

Import the parts, and compose them together.

```tsx title="kanban-usage.tsx"
import {
  Kanban,
  KanbanBoard,
  KanbanColumn,
  KanbanColumnHandle,
  KanbanItem,
  KanbanItemHandle,
  KanbanOverlay,
} from "@/components/ui/kanban";

return (
  <Kanban>
    <KanbanBoard>
      <KanbanColumn>
        <KanbanColumnHandle />
        <KanbanItem>
          <KanbanItemHandle />
        </KanbanItem>
      </KanbanColumn>
    </KanbanBoard>
    <KanbanOverlay />
  </Kanban>
)
```

## Usage [#usage]

### With Primitive Values [#with-primitive-values]

When using primitive arrays (strings, numbers), the `getItemValue` prop is optional. The component will automatically use the items themselves as unique identifiers.

```tsx title="kanban-example.tsx"
const [columns, setColumns] = React.useState({
  todo: ["Task 1", "Task 2"],
  done: ["Task 3"],
});

<Kanban.Kanban value={columns} onValueChange={setColumns}>
  <Kanban.Board>
    {Object.entries(columns).map(([columnId, items]) => (
      <Kanban.Column key={columnId} value={columnId}>
        {items.map((item) => (
          <Kanban.Item key={item} value={item}>
            {item}
          </Kanban.Item>
        ))}
      </Kanban.Column>
    ))}
  </Kanban.Board>
</Kanban.Kanban>
```

### With Object Arrays [#with-object-arrays]

When using object arrays, the `getItemValue` prop is required to extract unique identifiers from each item.

```tsx title="kanban-example.tsx"
const [columns, setColumns] = React.useState({
  todo: [
    { id: 1, title: "Task 1" },
    { id: 2, title: "Task 2" },
  ],
  done: [
    { id: 3, title: "Task 3" },
  ],
});

<Kanban.Kanban
  value={columns}
  onValueChange={setColumns}
  getItemValue={(item) => item.id}
>
  <Kanban.Board>
    {Object.entries(columns).map(([columnId, items]) => (
      <Kanban.Column key={columnId} value={columnId}>
        {items.map((item) => (
          <Kanban.Item key={item.id} value={item.id}>
            {item.title}
          </Kanban.Item>
        ))}
      </Kanban.Column>
    ))}
  </Kanban.Board>
</Kanban.Kanban>
```
